Introduction and Objectives

What our project is about?

The Russo-Ukrainian War is an ongoing conflict between Russia and Ukraine that began in 2014. The war has had a significant impact on both countries, with thousands of people killed and millions displaced. In this presentation, we will analyze the conflict using descriptive statistics to see some patterns and trends in the data.

So we decided to scroll Kaggle and find a dataset that would help us to analyze the conflict in recent years.

Data Description

Data Description

As it was mentioned before, we found some datasets on Kaggle. The dataset we chose contains information about the conflict in Ukraine from 2014 to 2021. It includes data on the number of casualties, attacks, and other relevant information.

So, the datasets:

  1. 2022 Russia Ukraine War (updated weekly)

  2. 2022 Russia Ukraine War, Losses, Oryx + Images

  3. Massive Missile Attacks on Ukraine

1. 2022 Russia Ukraine War

This dataset describes Equipment Losses & Death Toll & Military Wounded & Prisoner of War of russians in 2022-2024 Ukraine russia War by each day.

All data are official and additionally structured by the author (Petro).

Data:

  • russia_losses_personnel.csv - contains Personnel Losses during the war.

  • russia_losses_equipment.csv - contains Equipment Losses during the war.

  • russia_losses_equipment_correction.csv - contains some data correction (dates: 2022-10-13, date: 2023-05-27).

2. 2022 Russia Ukraine War, Losses, Oryx + Images

The dataset describes russian and Ukrainian Equipment Losses During The 2022 Russian Invasion Of Ukraine.

The dataset was created based on Oryx by scraping Ukrainian losses and russian losses pages.

Note

The datasets mentioned above are updated regularly and provide a comprehensive overview of the ongoing conflict. It is important to cross-verify the data with other sources for accuracy and reliability.

Warning!

This list only includes destroyed vehicles and equipment of which photo or videographic evidence is available. Therefore, the amount of equipment destroyed is significantly higher than recorded!

3. Massive Missile Attacks on Ukraine

The dataset contains available information about launched and shot down missiles and drones during russian massive missile and drone (UAV) strikes on infrastructure (since October 2022) as part of its invasion of Ukraine. The dataset also contains information about some single shot-down UAVs.

Data:

  • time_start - start attack time.
  • time_end - end attack time.
  • model - missile or UAV type.
  • launch_place - city or region from missiles were launched.
  • target - could be a city in Ukraine, or region of Ukraine or direction, or full Ukraine.
  • carrier - missile launch platform.
  • launched - number of launched missiles or UAVs.
  • destroyed - number of destroyed missiles or UAVs.
  • source - information source (mainly facebook posts).

    )

    for (col in corrected_columns) {
        correction_col <- paste0(col, "_correction")
        if (correction_col %in% colnames(corrected_data)) {
            # replace NA in correction with 0 (на всяк випадок)
            corrected_data[[correction_col]][is.na(corrected_data[[correction_col]])] <- 0
            # cumulative correction
            corrected_data[[col]] <- corrected_data[[col]] + cumsum(replace_na(corrected_data[[correction_col]], 0))
        }
    }
}

apply_corrections <- function(main_data, corrections) {
    corrected_data <- main_data %>%

    corrected_columns <- c(
      "aircraft", "helicopter", "tank", "APC", "field_artillery",
      "MRL", "drone", "naval_ship", "submarines", 
      "anti_aircraft_warfare", "special_equipment", 
      "vehicles_and_fuel_tanks", "cruise_missiles"
    )

    for (col in corrected_columns) {
        correction_col <- paste0(col, "_correction")
        if (correction_col %in% colnames(corrected_data)) {
            # replace NA in correction with 0 (на всяк випадок)
            corrected_data[[correction_col]][is.na(corrected_data[[correction_col]])] <- 0
            # cumulative correction
            corrected_data[[col]] <- corrected_data[[col]] + cumsum(replace_na(corrected_data[[correction_col]], 0))
        }
    }
    corrected_data <- corrected_data %>%
        select(-ends_with("_correction"))

    return(corrected_data)
}

categorize_missile <- function(missile) {
  severity_scores <- c(
    "High-powered missiles" = 10,
    "Medium-powered missiles" = 7,
    "Drones" = 3,
    "Composite weapon types" = 5,
    "Low-powered missiles / Others" = 2,
    "Unknown" = NA_real_  # Default for unknown categories
  )
  
  if (grepl("3M22 Zircon|Iskander-M|Iskander-K|X-47 Kinzhal|Kalibr|X-101/X-555|X-22|P-800 Oniks|X-59", missile, ignore.case = TRUE)) {
    return(severity_scores["High-powered missiles"])
  }
  else if (grepl("C-300|C-400|C-300/C-400|C-300/C-400 and Iskander-M|C-400 and Iskander-M|Iskander-M/KN-23|X-22/X-31P|X-22/X-32|X-31PD|X-31|KAB", missile, ignore.case = TRUE)) {
    return(severity_scores["Medium-powered missiles"])
  }
  else if (grepl("Shahed-136/131|Lancet|Mohajer-6|Orion|Orlan-10|Orlan-30|ZALA|Supercam|Merlin-VR|Eleron|Reconnaissance UAV|Unknown UAV", missile, ignore.case = TRUE)) {
    return(severity_scores["Drones"])
  }
  else if (grepl("Orlan-10 and Supercam|Orlan-10 and ZALA|Shahed-136/131 and Lancet|Orlan-10 and ZALA and Supercam|Orlan-10 and Orlan-30 and ZALA and Supercam", missile, ignore.case = TRUE)) {
    return(severity_scores["Composite weapon types"])
  }
  else if (grepl("Aerial Bomb|Grad/Urgan Rockets|Granat-4|Forpost|Картограф|Привет-82|Фенікс", missile, ignore.case = TRUE)) {
    return(severity_scores["Low-powered missiles / Others"])
  }
  
  return(severity_scores["Unknown"])  
}

Kvitka’s Hypothesis

The missile attacks are more severe on significant days?

The severity level of missile attacks is determined based on the type of weapon used, categorized into six groups:

  • High-powered missiles
  • Medium-powered missiles
  • Drones
  • Composite weapon types
  • Low-powered missiles / Others
  • Unknown

The missile attacks are more severe on significant days?

Each category is assigned a predefined score reflecting its destructive potential, with high-powered missiles scoring the highest and drones or low-powered weapons scoring lower.

This scoring system standardizes the assessment of attack intensity, enabling comparative analysis across different events or time periods.

But there is one big problem. The data is not normally distributed :(

So, I decided to use Bootstrap to compare the means of equipment losses in Summer 2023 and Autumn 2024.

bootstrap_normal <- function(data, n = 10000) {
  boot_means <- replicate(n, {
    sample(data, size = length(data), replace = TRUE) %>%
      mean()
  })
  boot_means
}

chi_squared_test <- function(data, mean, sd, breaks) {
  hist_data <- hist(data, breaks = breaks, plot = FALSE)
  observed <- hist_data$counts
  
  n <- sum(observed)
  
  expected <- numeric(length(observed))
  for (i in seq_along(expected)) {
    if (i == 1) {
      # Smallest bin
      expected[i] <- pnorm(breaks[i + 1], mean = mean, sd = sd) * n
    } else if (i == length(expected)) {
      # Largest bin
      expected[i] <- (1 - pnorm(breaks[i], mean = mean, sd = sd)) * n
    } else {
      # Middle bins
      expected[i] <- (pnorm(breaks[i + 1], mean = mean, sd = sd) -
                      pnorm(breaks[i], mean = mean, sd = sd)) * n
    }
  }
  
  # Combine bins with expected frequencies < 5
  while (any(expected < 5)) {
    idx <- which.min(expected)
    # if index is 1 or last, combine with next or previous bin
    if (idx == 1) {
      # Combine with next bin
      observed[2] <- observed[2] + observed[1]
      expected[2] <- expected[2] + expected[1]
      observed <- observed[-1]
      expected <- expected[-1]
    } else if (idx == length(expected)) {
      # Combine with previous bin
      observed[length(expected) - 1] <- observed[length(expected) - 1] + observed[length(expected)]
      expected[length(expected) - 1] <- expected[length(expected) - 1] + expected[length(expected)]
      observed <- observed[-length(expected)]
      expected <- expected[-length(expected)]
    } else {
      # Combine with smaller of adjacent bins
      if (expected[idx - 1] < expected[idx + 1]) {
        observed[idx - 1] <- observed[idx - 1] + observed[idx]
        expected[idx - 1] <- expected[idx - 1] + expected[idx]
        observed <- observed[-idx]
        expected <- expected[-idx]
      } else {
        observed[idx + 1] <- observed[idx + 1] + observed[idx]
        expected[idx + 1] <- expected[idx + 1] + expected[idx]
        observed <- observed[-idx]
        expected <- expected[-idx]
      }
    }
  }
  
  chi_sq_stat <- sum((observed - expected)^2 / expected)
  df <- length(observed) - 1 - 2  # we estimate mean and sd
  p_value <- pchisq(chi_sq_stat, df, lower.tail = FALSE)
  critical_value <- qchisq(0.95, df)
  
  cat("Chi-Squared Statistic:", chi_sq_stat, "\n")
  cat("Degrees of Freedom:", df, "\n")
  cat("P-value:", p_value, "\n")
  cat("Critical Value (alpha = 0.05):", critical_value, "\n")
  
  if (p_value < 0.05) {
    cat("Reject the null hypothesis: The data does not follow the assumed normal distribution.\n")
  } else {
    cat("Fail to reject the null hypothesis: The data follows the assumed normal distribution.\n")
  }
}

Conclusion

Our analysis of the Russo-Ukrainian War data, using various statistical methods, provides significant insights into the patterns of equipment and personnel losses as well as the dynamics of missile attacks. Here’s a summary of our findings:

1. Kvitka’s Hypothesis

  • Hypothesis: Missile attacks are more severe on significant dates.
  • Result: Statistical testing (t-tests) revealed no evidence to support the hypothesis. The severity of missile attacks appears independent of significant dates.
  • Implications: Other factors, such as military strategy or available resources, may influence the intensity of missile attacks rather than symbolic dates.

2. Zakhar’s Hypothesis

  • Hypothesis: Shelling points (missiles launched) are positively associated with equipment and personnel losses.
  • Model 1 (Equipment Losses):
    • Each additional shelling point increases equipment losses by ~24.46 points.
    • R²: 13.09% variability explained.
  • Model 2 (Personnel Losses):
    • Each additional shelling point increases personnel losses by ~6.30 points.
    • R²: 22% variability explained.
  • Implications: While the relationships are statistically significant, a substantial portion of the variability remains unexplained, highlighting the influence of other battlefield conditions.

3. Denys’s Hypothesis

  • Hypothesis I: Significant differences exist between equipment losses in Summer 2023 (Ukrainian counterattacks) and Autumn 2024.
  • Result: Bootstrapped t-tests revealed a significant difference between the two periods, supporting the hypothesis that loss rates in Summer 2023 (Ukrainian counterattacks) were higher.
  • Hypothesis II: The distribution of equipment losses in Summer 2023 follows a normal distribution.
  • Result: A Chi-Square Goodness-of-Fit test confirmed the hypothesis, as the test failed to reject the null hypothesis, indicating that the data aligns with a normal distribution.

This finding suggests that the Ukrainian counterattacks during Summer 2023 resulted in higher equipment losses compared to Autumn 2024, while the statistical distribution of losses during Summer 2023 is consistent with a normal pattern. These results provide further evidence of the intensity of operations during key periods of the conflict.

Final Insights

  • Statistical models confirm the relationships between shelling points and losses, but they also emphasize the need for further exploration of additional factors.
  • Seasonal and temporal patterns reveal no significant deviations in equipment losses during key periods like counterattacks.
  • Our findings suggest a balance between statistical rigor and the understanding that war data is influenced by myriad complex and often unpredictable factors.
